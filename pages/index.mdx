# Claude Code Cookbook

欢迎来到 Claude Code 使用教程。本站会带你从零上手 Claude Code，并通过一个可交互的终端模拟器让你直接在浏览器里感受它的使用方式。

## 本教程涵盖

- **快速开始** — 安装、登录、第一次对话
- **常用命令** — `/init`、`/review`、自定义 slash command
- **进阶** — hooks、MCP、agent、自定义 workflow
- **交互演练** — 在浏览器里体验 Claude Code 的交互

从 [快速开始](/getting-started) 或 [交互演练](/playground) 进入。

# 快速开始

## 安装

```bash
npm install -g @anthropic-ai/claude-code
```

## 登录

```bash
claude
```

首次运行会引导你完成认证。

## 第一次对话

在任意项目目录下执行 `claude`，然后描述你想做的事，比如：

> 帮我看看这个仓库的结构，并在 README 里加一段"本地开发"说明。

Claude Code 会读取相关文件、提出修改、并在你确认后写入磁盘。

## 下一步

- 试试内置命令：`/init`、`/review`、`/help`
- 打开 [交互演练](/playground) 在浏览器里感受一下
